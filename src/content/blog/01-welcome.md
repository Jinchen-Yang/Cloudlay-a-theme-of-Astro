# 欢迎来到我的博客

这是我使用 Astro 创建的个人博客。

## 为什么选择 Astro？

- 🚀 **高性能** - 默认零 JavaScript，页面加载速度快
- 📝 **灵活** - 支持 Markdown、React、Vue 等
- 🎨 **易于定制** - 创建自己的主题很容易
- ⚡ **开发体验好** - 快速的热更新

## 接下来

你现在可以：
1. 在 `src/content/blog/` 目录下创建新的 `.md` 文件
2. 添加文章的前置元数据（title、description 等）
3. 编写 Markdown 内容
4. 文章会自动显示在博客页面上

祝你开发愉快！

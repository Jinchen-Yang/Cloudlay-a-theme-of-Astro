## 📐 响应式设计细节

### 断点设置

| 设备 | 宽度 | 布局 | 侧边栏宽度 |
|-----|------|------|----------|
| **手机** | < 480px | 单列堆叠 | 100% |
| **大手机** | 480-768px | 单列堆叠 | 100% |
| **平板** | 768-1024px | 两列 | 280px |
| **桌面** | > 1024px | 两列 | 320px |

### 关键特性

#### 🖥️ 桌面版本 (> 1024px)
```
左侧侧边栏 (sticky, 320px) | 右侧内容区域
- 作者头像 (120px)          | - 欢迎区域
- 作者信息                  | - 文章列表
- 社交链接                  |   - 卡片1
- 统计数据                  |   - 卡片2
- 快速导航                  |   - 卡片3...
```

#### 📱 平板版本 (768px - 1024px)
```
侧边栏 (280px) | 内容区域
网格排列侧边栏 | 单列内容
```

#### 📱 手机版本 (< 768px)
```

内容区域
```

## 🎨 视觉设计

### 色彩方案

**主配色**：
- 主色: `#667eea` (紫蓝)
- 辅色: `#764ba2` (紫色)
- 背景: `#ffffff` (浅色) / `#0f0f0f` (深色)
- 文本: `#1a202c` (浅色) / `#e2e8f0` (深色)

**设计亮点**：
- 渐变背景卡片
- 发光效果头像
- 平滑的过渡动画
- 悬停提升效果

### 交互效果

```css
/* 卡片悬停 */
.post-card:hover {
  transform: translateY(-4px);        /* 向上平移 */
  box-shadow: 0 10px 25px rgba(...); /* 增强阴影 */
  border-color: #667eea;             /* 边框变主色 */
}

/* 按钮悬停 */
.nav-button:hover {
  transform: translateY(-2px);       /* 轻微上抬 */
  box-shadow: var(--shadow-lg);      /* 增加阴影 */
}

/* 社交链接 */
.social-link:hover {
  transform: translateY(-4px) scale(1.1);  /* 上抬+放大 */
}
```

## 🔧 自定义指南

### 快速自定义

1. **修改作者信息**
   ```astro
   // src/pages/index.astro
   const authorInfo = {
     name: "你的名字",
     title: "你的职位",
     avatar: "/avatar.jpg",
     // ... 其他信息
   };
   ```

2. **修改颜色主题**
   ```css
   /* src/pages/index.astro 中的 :root */
   :root {
     --color-primary: #667eea;  /* 改为你的主色 */
     --color-secondary: #764ba2; /* 改为你的辅色 */
   }
   ```

3. **修改头像**
   - 将图片放在 `/public/avatar.jpg`
   - 或修改 `authorInfo.avatar` 路径

4. **添加社交链接**
   - 编辑 `AuthorCard` 组件中的社交链接部分
